# RuboCop::Bitjourney

Custom cop for bitjourney projects.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rubocop-bitjourney'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubocop-bitjourney

## Usage

Add this line to your .rubocop.yml.

```yaml
require:
  - rubocop-bitjourney
```
